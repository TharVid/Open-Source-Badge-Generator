Frequently asked questions

Is the Open Source Badge Generator free?

Yes — 100% free. No signup, no credit card, no ads. It's an open source project hosted on GitHub Pages.

What data does the GitHub badge include?

Your display name, handle, avatar, bio, follower count, public repo count and total stars across all your repositories — plus a short appreciation note. Clean and easy to share.

Can I share the badge on LinkedIn and WhatsApp?

Yes. The Share panel has one-tap buttons for LinkedIn, WhatsApp, X (Twitter), Facebook, Telegram, Reddit and email, plus a native share sheet on mobile.

How do I download the badge as an image?

Click Download PNG. We render a 2× resolution image of the badge card you can post anywhere.

Does it work on mobile?

Absolutely. The UI is mobile-first and the native share sheet is supported on iOS and Android browsers that implement the Web Share API.

Why am I seeing a rate-limit error?

GitHub allows 60 unauthenticated API requests per IP per hour. If you hit the limit, wait an hour and try again.

Can I generate a badge for someone else?

Yes — any public GitHub username works. Share the resulting link (e.g. ?user=torvalds) with them.
